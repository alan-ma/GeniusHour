Introduction

Alan Ma

I'm a grade 11 student at the Woodlands School. This project is for my grade 11 physics class, taught by Mr. Fong.

The Question:

Is it possible to recreate real life physics in a game that incorporates a physical input system?

The Plan:

My goal is to model movement in the real world in a platform-style game. The goal is to create a peripheral device that allows the user to physically input data into the game, and show an object's acceleration, velocity, and displacement. The goal is for myself, a complete beginner at game physics, to bring computer science and physics together.

The Blog:

I want to be able to communicate my progress by showing code snippets, pictures, and video of the on-going project.
